# 多语言作品集网站

一个支持中英文切换的个人作品集网站，展示Python开发者的技能和项目经验。

## 功能特点

- **多语言支持**: 支持中英文一键切换，所有文本内容都已国际化
- **响应式设计**: 适配各种设备屏幕尺寸
- **现代化UI**: 采用渐变色彩和卡片式布局
- **项目展示**: 包含3个主要项目的详细介绍和演示链接
- **联系表单**: 功能完整的联系表单，支持用户留言

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **多语言**: 自定义i18n.js国际化方案
- **部署**: 支持静态文件部署

## 文件结构

```
lab04/
```

## 多语言功能

网站实现了完整的多语言切换功能：

1. **语言切换按钮**: 位于导航栏右侧，支持中文/英文切换
2. **翻译内容**: 所有文本内容都已添加`data-i18n`属性
3. **本地存储**: 自动保存用户语言偏好
4. **浏览器检测**: 根据浏览器语言自动设置默认语言

### 翻译键值结构

```javascript
// 翻译文件结构 (i18n.js)
translations = {
    zh: {
        nav: { home: "首页", about: "关于我", projects: "我的项目", contact: "联系我" },
        hero: { title: "张开发", subtitle: "Python开发者", cta: "查看我的作品" },
        about: { title: "关于我", description1: "...", description2: "..." },
        // ... 更多翻译内容
    },
    en: {
        // 英文翻译内容
    }
}
```

## 使用说明

1. **本地运行**:
   ```bash
   # 使用Python HTTP服务器
   python -m http.server 8000
   
   # 或使用Node.js http-server
   npx http-server
   ```

2. **语言切换**:
   - 点击导航栏右上角的语言按钮切换中英文
   - 系统会自动保存您的语言偏好

3. **项目演示**:
   - 点击项目卡片中的"查看演示"按钮
   - 每个项目都有独立的演示页面
